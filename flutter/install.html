# 安装flutter sdk

1.flutter sdk下载并解压。

2.添加path环境变量将flutter\bin的全路径作为它的值。

3.因为开发flutter需要联网使用flutter镜像国内因为墙的愿意需要配置一个国内镜像地址,将以下环境变量添加到用户环境变量中

 PUB_HOSTED_URL=https://pub.flutter-io.cn
 FLUTTER_STORAGE_BASE_URL=https://storage.flutter-io.cn

# 安装Android Studio下载。

一直next即可，虚拟机默认安装在C盘修改

# 安装虚拟机

打开Android Studio 工具栏 Tools-> AVD Manager -> create Virtul Device

可在用户环境变量中添加ANDROID_SDK_HOME = 需要安装的路径

TIP

启动虚拟机报错VT-x is disabled in BIOS。 重启电脑进入bios 将Intel Virtualization Technology开启即可

Android sdkmanager tool not found

# 检查安装时否正确。

可以在cmd运行 flutter doctor

TIP

Flutter plugin not installed; this adds Flutter specific functionality.需要在Android Studio安装flutter插件

Android license status unknown. 执行flutter doctor --android-licenses 如果出现Android sdkmanager tool not found 可以打开“Android studio” Tools-> SDK Manager -> SDK Tools ->取消选中Hide Obsolete Packages ->选中Android SDK Tools(Obsolete)->点击Apply安装即可

# vscode 开发 flutter

安装 Flutter 插件即可

新建项目Ctrl + Shift + P 输入Flutter: New Project

右下角 No Device点击即可看到安装的虚拟机列表，选择打开虚拟机

在终端输入 flutter run 即可运行创建的flutter项目

修改代码保存之后需要在将鼠标放在终端按R键即可编译到虚拟机或者打开调试模式实时重载
